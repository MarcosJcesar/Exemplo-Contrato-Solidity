# Exemplo-Contrato-Solidity
Esse trabalho representa didaticamente um contrato no estilo licitação pregão de menor preço, para fins de estudos academicos, abistraindo muito
conceitos da linguagem solidity para melhor compreensão de inciantes.
Para testa-lo, abrar o projeto Remix do Ethereum em https://remix.ethereum.org, clique em File explores na aba lateral,
clique botão direito do mouse em em contracts, escolha New File,
digite o nome do arquivo Licitacao1.sol, copie o cole o código na tela central que aparece.
Na aba lateral clique em Solidity Compiler, depois no botão azul Compile,(Não se esqueça de clicar antes no arquivo que criou)
Em Deploy & run transactions ainda na aba lateral você poderá fazer sua implantação em uma rede de teste do Remix simulando um sistema
blockchain em seu computador.
Tendo clicado em compile na aba Solidity compiler, vá para aba Deploy & run transactions e clique em escolha a primeira conta
que o Remix cria para testa e faça o deploy com ela.
Utilize JavaScript VM (London) o no campo "Environment", que é a aplicação que vai simular o blockchain do Ethereum na sua maquina para
que seja testado o contrato.
Tudo certo na parte de baixo da aba lateral esquerda irá aparecer os campos para seja testo o contrato.
Então é só testá-lo usando as instruições escritas no próprio código precedidas de duas barras "//"
